About DevMental Software

We build developer and engineering tools with .NET.

DevMental Software LLC specializes in building desktop applications using .NET and Avalonia UI for cross-platform deployment on Windows and Linux.

Our team brings over 15 years of software consulting experience across industries including aerospace engineering and developer tooling. We focus on performance-critical applications where every pixel and millisecond matters.

AI-Assisted Development

We are early adopters of AI-assisted software development, using Claude Code as a core part of our development workflow. Our flagship product DMEdit was built from scratch in approximately five weeks through pair programming with AI — demonstrating that a solo developer with the right tools can build software that would traditionally require a much larger team.

Our Products

  • DMEdit — A lightweight, fast, cross-platform text editor designed for large files
  • ENT — Engineering Note Tool for creating engineering analysis documents (ENT2 in development)
